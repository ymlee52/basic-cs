# 01-snake — 교재

> 하루에 한 페이지씩, 책 넘기듯이.

스네이크 게임을 만들면서 C와 CS 개념을 배우는 교재. 각 `stage-N.md`가 **하루치 분량**이고, 그 안에 그날의 코드와 해설이 모두 담겨 있다.

## 진행 방법

1. **읽기.** 페이지를 처음부터 끝까지 한 번 통독한다. 개념을 머리에 담는 게 먼저.
2. **클론코딩.** 페이지 안의 코드 블록을 그대로 타이핑한다. 복붙 X. 손가락에 새기는 게 목적.
3. **빌드 & 실행.** 페이지 끝의 "돌려보기" 절차대로.
4. **주석 곱씹기.** 동작이 확인되면 코드의 주석을 다시 읽어본다. 이때 "왜"가 들어와야 함.
5. **체크 질문.** 페이지 끝의 질문에 **소리 내어** 답해본다. 막히면 본문으로 돌아가서 다시.
6. **`NOTES.md` 갱신.** 새로 본 문법 / 깨달은 CS 개념 / 삽질이 있었다면 짧게.
7. **커밋.** 메시지에 "왜"를 짧게. 예: `stage 2: 키 입력으로 한 칸 이동 — stdin 버퍼링과 ANSI escape`

## 환경

- OS: Windows 11
- 컴파일러: `gcc` (MSYS2, 15.2.0)
- 빌드: `gcc 01-snake/snake.c -o 01-snake/snake`
- 실행: 터미널에서 `./01-snake/snake` (탐색기에서 더블클릭 X — 창이 곧바로 닫힘)
